1. Откройте настройки сети/прокси
2. Укажите:
   - **Host:** `localhost` или `85.209.135.247`
   - **Port:** `8082`
   - **SSL:** Включить
3. Установите сертификат mitmproxy (первый запуск)

## Как добавить операцию

1. Откройте панель: http://localhost:8083/admin
2. Заполните форму:
   - **Направление:** В другой банк (расход) / Из другого банка (доход)
   - **Сумма:** например, `1500`
   - **Банк:** выберите из списка (Сбербанк, ВТБ, Альфа и т.д.)
   - **Имя:** например, `Иван И.`
   - **Категория:** по умолчанию `Переводы`
   - **Дата и время:** опционально (по умолчанию сейчас)
3. Нажмите **"Добавить"**

Операция появится в списке, статистика обновится автоматически.

## API

### GET /api/operations
Получить список операций и статистику.

### POST /api/operations/add
Добавить операцию.
```json
{
  "direction": "out",
  "amount": 1500,
  "bank": "ВТБ",
  "title": "Перевод",
  "subtitle": "Переводы",
  "bank_preset": "vtb"
}
```

### POST /api/operations/delete
Удалить операцию.
```json
{"id": "m_xxx"}
```

### POST /api/operations/update
Обновить операцию.
```json
{
  "id": "m_xxx",
  "amount": 2000,
  "direction": "out",
  "title": "Новое имя"
}
```

### POST /api/operations/generate_receipt
Сгенерировать PDF-чек.
```json
{"id": "m_xxx"}
```

### POST /api/config/save
Сохранить конфиг.
```json
{
  "balance": {"new_balance": 99999.99},
  "name": {"first_name": "ИВАН"}
}
```

### POST /api/set_manual_stats
Установить ручную статистику.
```json
{"income": 100000, "expense": 50000}
```

## Примечания

- Операции отображаются, если дата за **текущий месяц** ИЛИ создана **менее 24 часов назад**
- Для генерации чеков нужны шрифты `TinkoffSans-Regular.ttf` и `TinkoffSans-Medium.ttf`
- Для генерации чеков нужен шаблон `sbpfinaltbanksend.pdf`
- Статистика считается автоматически, но можно установить вручную через панель
